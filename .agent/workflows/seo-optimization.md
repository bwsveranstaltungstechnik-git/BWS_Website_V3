# SEO Optimization Workflow

1. **Meta-Tags:**
   - Title Tags (max 60 Zeichen) pro Seite prüfen.
   - Meta Descriptions (max 160 Zeichen) optimieren.
2. **Keywords:**
   - Fokus-Keywords in H1 und Einleitungstexten sicherstellen.
3. **Performance:**
   - Bildgrößen prüfen.
   - Lazy Loading für Bilder am Ende der Seite aktivieren.
4. **Strukturierte Daten:**
   - Schema.org (Product/Article) für Olivenöl-Produkte integrieren.
