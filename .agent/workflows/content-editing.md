# Content Editing Workflow

1. **Inhaltsprüfung:**
   - Bestehende Texte auf Aktualität und Rechtschreibung prüfen.
   - Preise und Verfügbarkeit abgleichen.
2. **Media Update:**
   - Bilder optimieren (WebP-Format bevorzugt).
   - Alt-Texte für Barrierefreiheit hinzufügen.
3. **Anpassung:**
   - Änderungen direkt im HTML/JS vornehmen.
   - Sicherstellen, dass das Layout stabil bleibt.
4. **Final Check:**
   - Vorschau im Browser.
   - Cross-Browser-Kompatibilität kurz prüfen.
