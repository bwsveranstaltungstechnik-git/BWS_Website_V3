# Website Generation Workflow

1. **Konzeption:**
   - Zweck der neuen Seite/Sektion definieren.
   - Bildmaterial (Stock oder generiert) planen.
2. **Struktur:**
   - HTML-Skelett mit semantischen Tags erstellen.
   - SEO-relevante Überschriften (H1-H3) setzen.
3. **Design:**
   - Bestehendes CSS nutzen oder neue Klassen in `index.css` hinzufügen.
   - Mobile Responsiveness sicherstellen.
4. **Verifizierung:**
   - Darstellung im Browser prüfen.
   - Links und Formulare testen.
