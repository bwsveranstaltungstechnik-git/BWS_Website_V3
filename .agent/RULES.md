# Website Agent Rules & Guidelines

Diese Regeln gelten für alle KI-Agenten, die an der **Website-Oliven-Oel** arbeiten.

## 1. Projekt Kontext
- Dies ist eine Premium-Website für den Verkauf und die Präsentation von hochwertigem Olivenöl.
- Zielgruppe: Gourmets, gesundheitsbewusste Käufer und Liebhaber mediterraner Küche.
- Design: Edel, minimalistisch, Fokus auf hochwertigen Bildern und Storytelling.

## 2. Git & Workflow
- **Main Branch:** Alle Änderungen direkt auf `main` (außer bei größeren Experimenten).
- **Push nach Task:** Nach jedem abgeschlossenen Task `git push origin main`.
- **Dateinamen:** Konsistente Benennung in Deutsch oder Englisch (nach Absprache), bevorzugt Englisch für Code und Deutsch für Content.

## 3. Tech Stack
- HTML, CSS, JavaScript (Vanilla bevorzugt).
- Fokus auf Performance und SEO.
