Produkt-Detailseite — Layout

Wie soll die Detailseite aufgebaut sein? (Technisch: eine Seite, die per ?id=produkt-id befüllt wird)

A

Bild links / Info rechts (Empfohlen)

Klassisches Shop-Layout. Groß Produktbild links, rechts: Name, Preis/Tag, Beschreibung, Menge-Auswahl, "In den Warenkorb".

B

Großes Hero-Bild oben, Info darunter

Bild nimmt volle Breite ein, darunter alle Infos in zwei Spalten. Mehr visuellen Impact, weniger kompakt.

C

Modal / Overlay (kein Seitenwechsel)

Klick auf ein Produkt öffnet ein großes Overlay auf der aktuellen Seite. Kein Laden einer neuen Seite — schneller, aber keine eigene URL pro Produkt.
